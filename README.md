Demoblaze is a demo e-commerce website that allows users to browse products, view product details, add items to the cart, and place orders online. The application simulates a real-time shopping experience with 
core e-commerce functionalities such as product listing, category navigation, cart management, and order placement. This website is used for testing purposes to validate functional behavior, usability, 
and overall application stability through both manual and automation testing.
